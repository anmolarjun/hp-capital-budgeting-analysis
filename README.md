# HP Capital Budgeting Analysis

## Overview
This project presents a capital budgeting analysis for HP by evaluating three potential investment locations: India, China, and Vietnam.

The analysis combines quantitative valuation techniques with qualitative strategic considerations to determine the most attractive investment opportunity.

## Objectives

- Compare investment opportunities across three countries
- Calculate project cash flows
- Estimate Weighted Average Cost of Capital (WACC)
- Perform Net Present Value (NPV) analysis
- Calculate Internal Rate of Return (IRR)
- Incorporate qualitative business factors
- Recommend the optimal investment decision

## Workbook Structure

- **Introduction** – Project background and objectives
- **Assumptions** – Key financial assumptions
- **WACC** – Cost of capital calculation
- **India** – Investment analysis
- **China** – Investment analysis
- **Vietnam** – Investment analysis
- **Qualitative Considerations** – Strategic comparison
- **Conclusion** – Final recommendation

## Financial Techniques Used

- Net Present Value (NPV)
- Internal Rate of Return (IRR)
- Weighted Average Cost of Capital (WACC)
- Discounted Cash Flow (DCF)
- Sensitivity to investment assumptions
- Strategic qualitative assessment
